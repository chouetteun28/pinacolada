# Pinacolada
AI 교차편집 프로젝트

## 소개
Pinacolada는 인공지능 교차편집 프로젝트입니다.
공연 영상의 URL만 입력하면 웹 상에서 영상을 다운로드해서 자동으로 Face mesh, pose estimation과 같은 네트워크를 이용하여 부드러운 카메라 전환을 가능하게 합니다.
